## Project Overview

The **Neighborhood Map** displays interesting restaurants in Gr√†cia neighborhood (Barcelona) and other places that surround them. The APP allows also filtering the restaurant by cousine type. 

### Getting started

Please, clone or download the project in your computer.

```git clone https://github.com/tjblanco/mysites.git```

## Install

* Install all project dependencies with `npm install`
* Start the server with `npm start` 

## Dependencies

* Immutable (3.8.1)
* React (16.3.2)
* React-dom (16.3.2)
* React-router-dom (4.3.1)
* React-bootstrap (0.30.7)
* Redux (3.6.0)



The app will run automatically in your local host. 

## External API

This project fetches data from:

* [Google Map API](https://developers.google.com/maps/documentation/javascript/tutorial)
* [Foursquare API](https://developer.foursquare.com/)
